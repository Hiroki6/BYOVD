# BYOVD Collection

A collection of BYOVD (Bring Your Own Vulnerable Driver) PoCs that demonstrate kernel-level exploitation techniques on Windows.

## Disclaimer

**For educational purposes only.** These tools are intended to help understand how vulnerable drivers can be exploited and how defensive mitigations work. Do not use against systems you do not own or have explicit permission to test. The author is not responsible for any misuse or damage caused by these tools.
